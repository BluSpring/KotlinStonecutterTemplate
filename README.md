# Naz's Kotlin Stonecutter Template
A quick mod template for multiloader and multiversioned Stonecutter mods of mine, just so I don't keep copying
the same files over and over again and having them diverge between codebases.

This template is not intended as a "Getting Started" project, and assumes you already have knowledge
behind Stonecutter, Gradle, and Minecraft modding in general. If you don't have either of these,
may god help you, cuz I sure won't.

Is this practical? No.<br>
Is this good? No.<br>
Is it functional? Kinda.

This template is legit made only for myself, with all the terribleness built right in. Good luck to those
trying to use it.

This template does not support LexForge 1.20.2+, and does not intend to either.
